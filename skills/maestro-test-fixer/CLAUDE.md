# Maestro Test Fixer

This skill helps debug and fix failing Maestro UI tests.

## Installation for Claude Code

Copy this skill folder to your project's `.claude/skills/` directory:

```bash
cp -r skills/maestro-test-fixer /path/to/project/.claude/skills/
```

Or symlink it:

```bash
ln -s /path/to/skills/maestro-test-fixer /path/to/project/.claude/skills/maestro-test-fixer
```

## Usage

When you have a failing Maestro test, read `SKILL.md` for the debugging workflow.

For common issues and solutions, see `references/common-issues.md`.
